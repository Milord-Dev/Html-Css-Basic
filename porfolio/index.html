About Me

Soy Ricardo Milord un desarrollador de software junior y estudiante en la UTP en Panamá. Me apasionan el diseño web, el desarrollo frontend y backend, así como la exploración de nuevas tecnologías. Siempre estoy en constante aprendizaje para mejorar mis habilidades y aportar valor a los proyectos en los que participo. Además, en mi tiempo libre, canalizo mi creatividad a través de la producción musical.

"Aprender, crear y mejorar — siempre en movimiento."

Plataforma de Reservas Turísticas

Tienda Online de Panadería

Desarrollo de un e-commerce básico con catálogo y carrito de compras, desarrollado con NestJS y MongoDB. Backend con API REST para la gestión de inventario y pedidos, con interfaz sencilla para clientes.

Sistema de Apuestas Deportivas

Desarrollo de una aplicación web que muestra eventos deportivos y permite simular apuestas, desarrollada con HTML, JavaScript y Fastify. Consumo de API externa para obtener resultados y cálculo de ganancias potenciales.

Sistema de Gestión de Inventario

Desarrollo de una aplicación web para la gestión de inventario, desarrollada con HTML, JavaScript y Fastify. Backend con API REST para la gestión de productos y stock, con interfaz sencilla para usuarios.

Sistema de Gestión de Clientes

Desarrollo de una aplicación web para la gestión de clientes, desarrollada con HTML, JavaScript y Fastify. Backend con API REST para la gestión de clientes y pedidos, con interfaz sencilla para usuarios.

Sistema de Gestión de Proyectos

Desarrollo de una aplicación web para la gestión de proyectos, desarrollada con HTML, JavaScript y Fastify. Backend con API REST para la gestión de proyectos y tareas, con interfaz sencilla para usuarios.
